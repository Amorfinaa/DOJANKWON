<h2>🛠️ Installation Steps:</h2>

<p>1. Clona el repositorio</p>

```
git clone https://github.com/Amorfinaa/DOJANKWON.git
```

<p>2. Instala las dependencias</p>

```
npm install
```

<p>3. Ejecuta la aplicación</p>

```
npm run dev
```
